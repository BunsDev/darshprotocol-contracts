## Development
- Compile contracts
`truffle compile`

- Migrate contracts
`truffle migrate --network=testnet|mainnet --compile-none`

- Set configs
`truffle test --network=testnet|mainnet --compile-none` -t1

- Run test
`truffle test --network=testnet|mainnet --compile-none` -f2
